---
description: "Show task progress (or all ACTIVE) / Показать прогресс задачи (или всех ACTIVE)"
argument-hint: [id]
---

Activate `swift-toolkit:task-status`.

$ARGUMENTS parsing:
- Non-empty — show progress for a specific task (task_id = $ARGUMENTS)
- Empty — show a compact table of all tasks in `Tasks/ACTIVE/`
